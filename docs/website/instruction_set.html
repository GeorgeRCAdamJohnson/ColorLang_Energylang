Instruction Set

The ColorLang instruction set defines the operations that can be encoded as HSV pixel values and executed by the Virtual Machine (VM). Each instruction corresponds to a specific hue range, with saturation and value providing additional operands or parameters.

Instruction Categories

  • Arithmetic Instructions: ADD, SUB, MUL, DIV, MOD, POW.
  • Memory Instructions: LOAD, STORE, MOVE, COPY, ALLOC, FREE.
  • Control Flow Instructions: IF, WHILE, FOR, BREAK, CONTINUE.
  • Function Instructions: CALL, RETURN, FUNC_DEF, PARAM, LOCAL.
  • I/O Instructions: PRINT, INPUT, READ_FILE, WRITE_FILE, NETWORK_SEND, NETWORK_RECV.
  • System Instructions: HALT, DEBUG, THREAD_SPAWN, THREAD_JOIN, MUTEX_LOCK, MUTEX_UNLOCK.

Encoding Details

Each instruction is encoded using HSV values:

  • Hue: Determines the instruction category.
  • Saturation: Encodes the first operand.
  • Value: Encodes the second operand.
