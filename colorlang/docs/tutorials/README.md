# ColorLang Tutorials

## Overview
This directory contains hands-on tutorials for learning ColorLang development. Each tutorial focuses on a specific aspect of the language and builds practical skills through examples.

- **[Debugging Techniques](debugging-techniques.md)** - Finding and fixing issues

## Tutorial Structure
Each tutorial follows a consistent format:
1. **Objectives** - What you'll learn
2. **Prerequisites** - Required knowledge
3. **Step-by-step Instructions** - Hands-on activities  
4. **Code Examples** - Working programs to try
5. **Exercises** - Practice problems
6. **Next Steps** - Where to go from here
