User Guide

Welcome to ColorLang! This guide will help you get started with writing and running ColorLang programs. Whether you're creating a simple "Hello World" or a complex AI-driven application, this guide will walk you through the basics.

Getting Started

  • Prerequisites: Python 3.12 or later and required libraries.
  • Writing a Program: Use an image editor or the examples/create_examples.py script to create programs.
  • Running a Program: Parse and execute programs using the ColorLang parser and VM.

Advanced Topics

  • Procedural Generation: Generate programs procedurally using scripts.
  • Shared Memory: Synchronize data between the VM and host applications.
  • Networking: Enable communication between machines using ColorLang instructions.
