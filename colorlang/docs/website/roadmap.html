Roadmap

ColorLang aims to revolutionize machine-native programming by enabling seamless communication between machines and AI systems through a visual, HSV-based language. The ultimate goal is to create a system that evolves from procedural logic to adaptive intelligence, eventually achieving AGI (Artificial General Intelligence) capabilities.

Version 1.0: Foundational System

  • Demonstrate ColorLang's ability to encode and execute programs, from simple tasks to complex applications.
  • Create a procedurally generated platformer game where a monkey plays automatically.
  • Enable communication between machines/AI across a network using ColorLang.

Version 2.0: Autonomous Learning Agent

  • Build an agent that can take in tasks via an API and learn from them.
  • Expand the ability to generate complex environments and tasks procedurally.

Version 3.0: Simulated Intelligence

  • Develop a simulated intelligence capable of reasoning and making decisions based on incomplete information.
  • Enable the system to analyze its own performance and refine its logic without external input.

Version 4.0: Artificial General Intelligence (AGI)

  • Achieve a small-scale AGI capable of reasoning, learning, and dreaming.
  • Mimic human neural processes by forming tighter connections between concepts through "color-based" reasoning.
